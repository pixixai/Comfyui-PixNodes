# IfElse 分发器

## 1. 节点功能概览

`Pix_IfElseDispatcher` 是一个逻辑流控制节点，能够根据输入的布尔值（Boolean）决定数据的流向。它的核心特性是利用 ComfyUI 的 `ExecutionBlocker` 机制，**物理性地阻塞**未选中分支的下游执行，从而实现真正意义上的逻辑分支切换，避免不必要的计算资源浪费。

## 2. 输入参数解释

| 参数名称 | 类型 | 建议值 | 描述 |
| --- | --- | --- | --- |
| **any_value** | * (任意) | 任何输出信号 | 需要被分发的原始数据或信号。支持所有 ComfyUI 类型。 |
| **if_true** | BOOLEAN | True / False | 控制开关。为 `True` 时数据流向真分支；为 `False` 时流向假分支。 |

## 3. 输出结果描述

| 索引 | 名称 | 类型 | 描述 |
| --- | --- | --- | --- |
| **0** | true_branch | * | 当 `if_true` 为真时，输出 `any_value`；否则下游被阻塞。 |
| **1** | false_branch | * | 当 `if_true` 为假时，输出 `any_value`；否则下游被阻塞。 |

## 4. 典型工作流场景

- **条件渲染**: 根据用户选择（如：是否开启高清修复）来决定图像是直接输出，还是进入放大采样流程。
- **动态资源切换**: 在不同的采样器或模型之间进行逻辑切换，确保只有被激活的路径会占用显存和计算资源。
- **安全过滤**: 配合逻辑判断节点，若检测到敏感内容，则切换到空白输出路径。
