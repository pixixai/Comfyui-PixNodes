# PixNodes 循环控制节点使用说明

本插件基于 ComfyUI V3 动态图扩展架构，提供了强大的 **For 循环** 和 **While 循环** 功能。支持动态插槽扩展。

## 核心节点概览

插件包含两组核心节点，分别对应两种常见的编程循环模式：

1. **For Loop (计数循环)**：指定循环次数，自动执行。
2. **While Loop (条件循环)**：基于布尔条件判断，条件为真时持续执行。

## For Loop Start (Pix_ForLoopStart)

**功能**：定义循环的起点和迭代次数。

### 输入参数

- **total (INT)**:
    - **含义**: 循环执行的总次数。
    - **范围**: 1 ~ 100000。
    - **说明**: 类似于编程中的 `for (int i=0; i < total; i++)`。
- **initial_value_1 (Any)**:
    - **含义**: 初始变量值。
    - **说明**: 连接任意类型的数据（图像、模型、数字等）。连接后会自动出现 `initial_value_2`，支持无限扩展。

### 输出结果

- **flow (FLOW_CONTROL)**:
    - **含义**: 控制流句柄。
    - **用法**: **必须**连接到对应的 `For Loop End` 节点的 `flow` 输入端。
- **index (INT)**:
    - **含义**: 当前循环的索引（从 0 开始）。
    - **用法**: 可用于计算进度、种子增量等。
- **value_1 (Any)**:
    - **含义**: 当前迭代的变量值。
    - **用法**: 在循环体内部使用该值进行处理。
