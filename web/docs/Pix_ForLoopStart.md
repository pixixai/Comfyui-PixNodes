# for循环开始

## For Loop Start (Pix_ForLoopStart)

**功能**：定义循环的起点和迭代次数。

### 输入参数

- **total (INT)**:
    - **含义**: 循环执行的总次数。
    - **范围**: 1 ~ 100000。
    - **说明**: 类似于编程中的 `for (int i=0; i < total; i++)`。
- **initial_value_1 (Any)**:
    - **含义**: 初始变量值。
    - **说明**: 连接任意类型的数据（图像、模型、数字等）。连接后会自动出现 `initial_value_2`，支持无限扩展。

### 输出结果

- **flow (FLOW_CONTROL)**:
    - **含义**: 控制流句柄。
    - **用法**: **必须**连接到对应的 `For Loop End` 节点的 `flow` 输入端。
- **index (INT)**:
    - **含义**: 当前循环的索引（从 0 开始）。
    - **用法**: 可用于计算进度、种子增量等。
- **value_1 (Any)**:
    - **含义**: 当前迭代的变量值。
    - **用法**: 在循环体内部使用该值进行处理。
