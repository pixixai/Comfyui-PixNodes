# Pix_EmptyListGenerator (空列表生成器) 使用指南

## 1. 节点功能概览

`EmptyListGenerator` 是一个功能单一但不可或缺的辅助节点。它唯一的作用是输出一个标准的 Python **空列表 `[]`**。

在 ComfyUI 的逻辑流中，该节点通常作为**累加器（Accumulator）**的起点，配合循环节点或列表追加节点（如 `List Append`）使用。

## 2. 输入与输出参数

### 输入 (Inputs)

- **无**: 该节点不需要任何输入即可运行。

### 输出 (Outputs)

- **list (LIST / ANY)**:
    - **描述**: 返回一个初始化后的空列表。
    - **技术细节**: 采用通配符类型 ，可以与绝大多数支持列表操作的节点连接。

## 3. 典型应用场景：循环数据收集

这是该节点最经典的使用方式，通常与 `Pix_ForLoop` 配合：

1. **初始化**: 在 `For Loop Start` 之前，连接一个 `Empty List Generator` 到 `initial_value_1`。
2. **循环体**:
    - 在循环内部执行采样或其他操作，得到一个结果（如 Image 或 String）。
    - 使用列表操作节点（如 `List Append`），将本轮结果追加到从 `For Loop Start` 传进来的列表中。
3. **反馈回路**: 将追加后的新列表连回 `For Loop End`。
4. **结果**: 当循环结束时，`For Loop End` 的输出端将得到一个包含所有轮次结果的完整列表。

## 4. 常见问题

- **为什么不直接在列表节点里写？**: 许多列表操作节点需要一个“基准列表”。如果没有初始化节点，在第一轮循环时由于没有输入源，工作流可能会报错或无法正确建立数据流。
