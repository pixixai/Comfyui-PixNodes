# Pix_BaiduTranslateNode (百度翻译) 节点说明

## 1. 节点功能概览

`Pix_BaiduTranslateNode` 是 PixNodes 系列中的文本处理节点，专门对接百度翻译开放平台。它常用于将中文提示词自动翻译为英文，或在多语言工作流中进行文本格式转换。

**核心特性**：

- **凭据持久化**：AppID 和 AppKey 填入运行一次后会自动保存至 `ComfyUI/user/PixNodes/api_key.json`，后续可留空。
- **自动检测**：支持源语言自动识别。
- **错误自愈**：如果检测到鉴权错误（如 Key 填错），节点会自动清除本地错误的缓存凭据，引导用户重新输入。

## 2. 输入参数解释

| 参数名称 | 类型 | 建议值 | 描述 |
| --- | --- | --- | --- |
| **appid** | STRING | (百度 AppID) | 百度翻译开放平台提供的 AppID。填入运行后可留空。 |
| **appkey** | STRING | (百度密钥) | 百度翻译开放平台提供的密钥。填入运行后可留空。 |
| **from_language** | 菜单 | 自动检测(auto) | 源文本的语言类型。 |
| **to_language** | 菜单 | 英文(en) | 目标翻译语言。若用于跑图，通常选择英文。 |
| **text** | STRING | (待翻译文字) | 需要进行翻译的原始文本，支持多行。 |

## 3. 输出结果描述

| 索引 | 名称 | 类型 | 描述 |
| --- | --- | --- | --- |
| **0** | translated_text | STRING | 翻译后的最终文本结果。如果发生 API 错误，将输出错误提示。 |

## 4. 典型工作流场景

- **提示词汉化/英化**：在输入框直接写中文 Prompt，通过该节点翻译后传给 `CLIP Text Encode`。
- **多语言文本处理**：配合 `Pix_LoadTextFromFolder` 读取本地多语言文档并统一翻译为指定语言进行分析。
- **自动化工作流说明**：将工作流的运行结果或 AI 的回复内容翻译成用户的母语显示。
