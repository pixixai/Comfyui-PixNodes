# Pix_ListToImageBatch (列表转图像批次) 节点说明

## 1. 节点功能概览

`Pix_ListToImageBatch` 属于 `PixNodes` 系列，主要用于将 ComfyUI 中的“图像列表”（List of Tensors）转换为标准的“图像批次”（Image Batch Tensor）。该节点具备智能兼容性，如果列表中的图像分辨率不一致，它会自动以第一张图像的尺寸为基准，对后续图像进行双线性插值（Bilinear）缩放。

## 2. 输入参数解释

| 参数名称 | 类型 | 建议连接 | 描述 |
| --- | --- | --- | --- |
| **image_list** | * (列表) | 任何输出图像列表的节点 | 接收一个包含多个图像 Tensor 的 Python 列表。 |

## 3. 输出结果描述

| 索引 | 名称 | 类型 | 描述 |
| --- | --- | --- | --- |
| **0** | image_batch | IMAGE | 合并后的标准图像批次。可直接连接到 `VAE Encode` 或 `Image Save` 等节点。 |

## 4. 典型工作流场景

- **循环收集器汇总**: 当使用循环节点（Loop）处理多张单图并收集到列表后，使用此节点将它们打包成一个批次进行统一保存或后期处理。
- **异构图像合并**: 当有多个来源、尺寸略有不同的图像需要强制合并为一个视频帧序列或批次时，此节点提供的自动缩放功能非常实用。
- **类型强制转换**: 将单个图像 Tensor（维度为 3）安全地转换为批次格式（维度为 4）。
