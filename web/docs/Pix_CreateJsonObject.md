# 创建 JSON 对象

**Pix_CreateJsonObject** 是一个提供富交互界面的节点，允许用户以类似 Notion 的“块”操作方式，轻松创建和编辑 JSON 对象（键值对）。

## 核心功能

### 1. 键值对编辑 (Key-Value)

- **Key (键)**：显示为深绿色，仅支持字符串。
- **Value (值)**：显示为灰色（选中时变白），支持多种数据类型。
- **自动保存**：所有的编辑操作都会实时序列化并保存到工作流中。

### 2. 块操作与快捷键

- **创建新块**：
    - 在 **Value** 输入框中按下 `Enter` 键，会在下方创建一个新块。
    - 点击底部的 **+** 胶囊按钮。
- **焦点切换**：
    - 在 **Key** 中按下 `Enter`，焦点跳转到对应的 Value。
    - 使用 `↑` 和 `↓` 方向键在不同的输入框之间快速导航。
- **删除与合并**：
    - 在 **Value** 开头按下 `Backspace`，焦点跳回 Key。
    - 在 **Key** 开头按下 `Backspace`，尝试跳转到上一行。
    - 悬停在块上，右上角会显示 **×** 删除按钮。

### 3. 高级选择与排序

此节点支持完整的文件管理器风格的选择逻辑：

- **单选**：点击块的任意位置（文字或边缘）。
- **加选 (Ctrl/Cmd + 点击)**：按住 Ctrl 点击，可逐个添加或取消选中块。
- **连选 (Shift + 点击)**：按住 Shift 点击，可选中两个点击点之间的所有块。
- **全选/复制**：支持 `Ctrl+A` 全选，`Ctrl+C` 复制为标准 JSON 文本。
- **取消选择**：点击列表下方的空白区域。

### 4. 拖拽排序

- 在**多选模式**下，按住任意选中块的**左侧边缘（手柄区域）**进行拖拽，可以同时移动所有选中的块，调整 JSON 对象的顺序。

### 5. 数据类型系统

支持四种数据类型，右键点击 **Value** 区域或边缘即可修改：

- **String** (默认)：普通文本。
- **Number**：数字，会自动转换格式。
- **Boolean**：布尔值 (true/false)。
- **JSON**：嵌套的 JSON 对象或数组。

## 输入与输出

### 输入

- **json_data (Hidden)**: 存储节点状态的序列化字符串，通常对用户隐藏。

### 输出

- **JSON 对象** :
    - 类型：`JSON` (Dict)
    - 说明：Python 原生字典对象。适合连接到其他数据处理节点进行逻辑运算。
