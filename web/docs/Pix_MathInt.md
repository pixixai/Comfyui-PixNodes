# 整数运算器

## 1. 节点功能概览

`Pix_MathInt` 是一个专门用于处理整数（Integer）的数学运算节点，属于 `PixNodes/Logic` 分类。它提供了标准的算术操作，常用于工作流中的动态参数计算、循环计数器处理以及逻辑判断前的数据预处理。

## 2. 输入参数解释

| 参数名称 | 类型 | 建议值 | 描述 |
| --- | --- | --- | --- |
| **a** | INT | 任意整数 | 运算的第一个操作数。 |
| **b** | INT | 任意整数 | 运算的第二个操作数。 |
| **operation** | 菜单 | `add`, `multiply` 等 | 选择要执行的数学运算类型。 |

### 运算类型详解：

- **add (加)**: $a + b$
- **subtract (减)**: $a - b$
- **multiply (乘)**: $a \times b$
- **divide (除)**: $a // b$ (整除，除数为0时返回0)
- **modulo (取模)**: $a \% b$ (求余数)
- **power (幂)**: $a^b$

## 3. 输出结果描述

| 索引 | 名称 | 类型 | 描述 |
| --- | --- | --- | --- |
| **0** | result | INT | 运算后的整数结果。 |

## 4. 典型工作流场景

- **动态步数计算**: 根据图像复杂度动态增加或减少采样的 `steps`。
- **批次编号处理**: 在循环处理中，利用 `add` 节点对序列号进行递增。
- **条件触发器**: 配合 `Compare` 节点。例如：使用 `modulo` 判断当前索引是否为偶数，从而决定是否执行某些特定操作。
