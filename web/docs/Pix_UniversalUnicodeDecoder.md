# Universal Unicode Decoder (通用 Unicode 解码器)

## 节点功能概览

`Universal Unicode Decoder` 是一个通用的数据清洗与解码工具。它专门设计用于处理 ComfyUI 流程中出现的 `\uXXXX` 格式的 Unicode 转义序列、JSON 转义字符以及控制字符（如换行符 `\n`、制表符 `\t`）。

该节点具备深度递归处理能力，能够自动穿透列表（List）、字典（Dict）及元组（Tuple）等复杂嵌套结构，将其中的所有编码文本还原为标准的中文字符，并最终统一输出为易于阅读的格式化字符串。

## 输入参数解释

### 必须输入 (Required)

- **input_data** ():
    - **类型**: 通配符（支持任何类型）。
    - **描述**: 接收待解码的原始数据。可以是简单的字符串，也可以是来自 API、数据库或循环累加器生成的复杂嵌套 JSON 对象/列表。

## 输出结果描述

- **text** (`STRING`):
    - **描述**: 解码后的标准 UTF-8 字符串。
    - **特性**:
        - 所有的 Unicode 编码（如 `\u6210`）将被还原为对应的中文字符。
        - 所有的转义控制符（`\n`, `\t`, `\r`）将被还原为真实的排版操作。
        - 如果输入是列表，将自动以换行符连接各元素。
        - 如果输入是复杂对象，将以美化后的 JSON 格式输出。

## 典型工作流应用场景

1. **API 响应清洗**: 处理从外部 LLM 或网络 API 返回的带有 Unicode 转义的 JSON 文本。
2. **循环日志预览**: 配合循环节点，将收纳在列表中的多条中文提示词转化为可直接阅读的文本预览。
3. **文本排版修正**: 修复由于数据传输导致的换行符失效问题，恢复文本的原始段落结构。
