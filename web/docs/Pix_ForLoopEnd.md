# PixNodes 循环控制节点使用说明

本插件基于 ComfyUI V3 动态图扩展架构，提供了强大的 **For 循环** 和 **While 循环** 功能。支持动态插槽扩展。

## 核心节点概览

插件包含两组核心节点，分别对应两种常见的编程循环模式：

1. **For Loop (计数循环)**：指定循环次数，自动执行。
2. **While Loop (条件循环)**：基于布尔条件判断，条件为真时持续执行。

## For Loop End (Pix_ForLoopEnd)

**功能**：定义循环的终点，并收集下一次迭代的数据。

### 输入参数

- **flow (FLOW_CONTROL)**:
    - **含义**: 接收来自 `For Loop Start` 的控制流。
- **initial_value_1 (Any)**:
    - **含义**: 本次迭代处理后的结果。
    - **用法**: 将循环体内处理完的数据连回此处。该值将作为下一次迭代的输入传回 Start 节点。

### 输出结果

- **value_1 (Any)**:
    - **含义**: 循环结束后的最终结果。
    - **说明**: 输出的是最后一次迭代完成后的数据。
