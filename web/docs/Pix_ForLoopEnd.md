# for循环结束

## For Loop End (Pix_ForLoopEnd)

**功能**：定义循环的终点，并收集下一次迭代的数据。

### 输入参数

- **flow (FLOW_CONTROL)**:
    - **含义**: 接收来自 `For Loop Start` 的控制流。
- **initial_value_1 (Any)**:
    - **含义**: 本次迭代处理后的结果。
    - **用法**: 将循环体内处理完的数据连回此处。该值将作为下一次迭代的输入传回 Start 节点。

### 输出结果

- **value_1 (Any)**:
    - **含义**: 循环结束后的最终结果。
    - **说明**: 输出的是最后一次迭代完成后的数据。
