# 图像对比

**Image Comparer (PixNodes)** 是一个用于在 ComfyUI 界面内直接对比两张或两组图像的实用节点。它支持滑动对比和点击切换对比两种模式，并且不依赖额外的复杂库。

## 输入参数 (Inputs)

该节点接受两个图像输入，分别作为前景和背景进行对比：

- **foreground_image (Optional)**: 前景图像。在节点上通常标记为 **F** (Foreground)。在滑动模式下，这通常显示在左侧（被裁剪覆盖在背景之上）。
- **background_image (Optional)**: 背景图像。在节点上通常标记为 **B** (Background)。在滑动模式下，这作为底图显示在右侧。

> 注意: 如果输入的是图像批次 (Batch)，节点顶部会自动显示 F1, F2... 和 B1, B2... 的选择器，点击即可切换当前对比的图片对。
> 

## 使用方法 (Usage)

### 1. 对比模式 (Comparer Mode)

你可以通过右键点击节点，在 **Properties** (属性) 面板中修改 `comparer_mode`，或者通过自定义的右键菜单项（如果有）来切换。

- **Slide (滑动模式 - 默认)**:
    - **交互**: 鼠标在节点上左右移动。
    - **效果**: 鼠标位置作为分割线。分割线左侧显示 **前景图 (F)**，右侧显示 **背景图 (B)**。这非常适合观察滤镜、重绘或放大前后的细节差异。
    - *提示*: 鼠标移出节点区域时，默认显示前景图。
- **Click (点击模式)**:
    - **交互**: 鼠标点击并按住节点区域。
    - **效果**:
        - **松开 (默认)**: 显示 **前景图 (F)**。
        - **按住**: 显示 **背景图 (B)**。
    - 这适合快速检查两张图的整体差异（类似于 "A/B Test"）。

### 2. 端口布局 (Connections Layout)

为了优化连线美观度，该节点支持动态调整输入/输出端口的位置。

- **操作**: 右键点击节点 -> 选择 `Connections Layout`。
- **选项**:
    - `Left -> Right`: 左进右出 (默认)
    - `Right -> Left`: 右进左出
    - `Top -> Bottom`: 上进下出
    - `Bottom -> Top`: 下进上出

## 输出 (Outputs)

- **无**: 该节点主要用于预览和交互，没有输出图像端口。但它被标记为 `OUTPUT_NODE`，因此可以作为工作流的终点运行。
