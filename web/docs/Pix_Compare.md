# Pix_Compare (逻辑比较器) 节点说明

## 1. 节点功能概览

`Pix_Compare` 是逻辑工作流中的“大脑”，属于 `PixNodes/Logic` 分类。它能够接收两个任意类型的数据（通常为数值或字符串），根据预设的数学逻辑进行比较，并输出一个布尔值（True/False）。它是触发 `IfElse 分发器` 等逻辑切换节点的必备前置节点。

## 2. 输入参数解释

| 参数名称 | 类型 | 建议值 | 描述 |
| --- | --- | --- | --- |
| **a** | * (任意) | 数值或字符串 | 比较运算的左侧操作数。 |
| **b** | * (任意) | 数值或字符串 | 比较运算的右侧操作数。 |
| **comparison** | 菜单 | `a == b` 等 | 选择比较逻辑：等于、不等于、大于、小于、大于等于、小于等于。 |

## 3. 输出结果描述

| 索引 | 名称 | 类型 | 描述 |
| --- | --- | --- | --- |
| **0** | boolean | BOOLEAN | 比较结果。如果条件成立返回 True，否则返回 False。 |

## 4. 典型工作流场景

- **自动化参数检查**: 比较当前步数是否达到了预设值，从而决定是否切换采样器。
- **动态分支切换**:
    - 输入 A：图像的宽度。
    - 输入 B：1024。
    - 比较条件：`a > b`。
    - 输出：将结果连至 `IfElse 分发器`，实现“大图”与“小图”走不同的放大路径。
- **文本内容判定**: 检查两个字符串是否相等，以实现简单的标签化逻辑过滤。
