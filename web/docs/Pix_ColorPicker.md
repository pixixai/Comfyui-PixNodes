# Pix_ColorPicker (拾色器) 节点说明

## 1. 节点功能概览

`Pix_ColorPicker` 是一个多功能的颜色处理工具，属于 `PixNodes` 系列。它允许用户通过多种常见的颜色格式（RGB、HSV、HEX）输入颜色，并将其自动转换为其他多种标准格式输出。该节点解决了 ComfyUI 中不同插件间颜色参数格式不统一的问题。

## 2. 输入参数解释

| 参数名称 | 类型 | 建议值/格式 | 描述 |
| --- | --- | --- | --- |
| **color_mode** | 菜单 | RGB, HSV, HEX | 选择当前输入的颜色模式。 |
| **value** | 字符串 | `255,128,64` (RGB)

`20,50,100` (HSV)

`#FF8040` (HEX) | 根据选择的模式填入对应颜色的数值。 |
- **RGB 模式**: 输入三个 0-255 之间的整数，用逗号分隔。
- **HSV 模式**: 输入格式为 `H(0-360), S(0-100), V(0-100)`。
- **HEX 模式**: 输入 6 位 16 进制颜色码（支持带或不带 `#` 号）。

## 3. 输出结果描述

| 索引 | 名称 | 类型 | 描述 |
| --- | --- | --- | --- |
| **0** | RGB | STRING | 标准 RGB 字符串，例如 `"255,128,64"`。 |
| **1** | RGB_INT | INT | RGB 的整数表示法，常用于需要颜色索引或位运算的节点。 |
| **2** | HSV | STRING | HSV 字符串，例如 `"20.0,50.0,100.0"`。 |
| **3** | HEX | STRING | 带 `#` 号的 16 进制字符串，例如 `"#FF8040"`。 |
| **4** | HEX_NO_HASH | STRING | 不带 `#` 号的 16 进制字符串，例如 `"FF8040"`。 |

## 4. 典型工作流场景

- **色彩统一管理**: 在复杂工作流中，作为全局颜色控制器，将一个 HEX 颜色值分发给需要 RGB 字符串的节点和需要整数颜色的节点。
- **动态调色**: 配合字符串拼接节点，根据特定逻辑动态生成颜色输入，并实时查看不同格式的转换结果。
