# 创建 JSON 列表

**Pix_CreateJsonList** 是一个提供富交互界面的节点，允许用户以有序列表的方式，轻松创建和编辑 JSON 数组。它继承了 JSON 对象节点的“块”操作体验，并针对列表逻辑进行了优化。

## 核心功能

### 1. 列表项编辑 (Ordered List)

- **Index (索引)**：显示为 `[ 0 ]`、`[ 1 ]` 格式，由系统自动维护，**不可编辑**。无论如何拖拽排序，索引总是从 0 开始依次递增。
- **Value (值)**：显示为灰色（选中时变白），支持多种数据类型。
- **自动保存**：所有的编辑操作都会实时序列化并保存到工作流中。

### 2. 块操作与快捷键

- **创建新行**：
    - 在 **Value** 输入框中按下 `Enter` 键，会在下方插入一个新行并自动聚焦。
    - 点击底部的 **+** 胶囊按钮。
- **焦点导航**：
    - 使用 `↑` 和 `↓` 方向键在不同的 **Value** 输入框之间快速切换。
- **智能删除**：
    - 在空行的 **Value** 中按下 `Backspace`，会删除当前行，并将焦点移动到上一行。
    - 悬停在块上，右上角会显示 **×** 删除按钮。

### 3. 高级选择与排序

支持完整的文件管理器风格选择逻辑，方便批量操作：

- **单选**：点击块的任意位置。
- **加选 (Ctrl/Cmd + 点击)**：按住 Ctrl 点击，可逐个添加或取消选中块。
- **连选 (Shift + 点击)**：按住 Shift 点击，可选中两个点击点之间的所有块。
- **批量复制**：支持 `Ctrl+C`，将选中的值复制为标准 JSON 数组格式（例如 `["Value1", "Value2"]`）。
- **取消选择**：点击列表下方的空白区域。

### 4. 拖拽排序

- 在**多选模式**下，按住任意选中块的**左侧边缘（手柄区域）**进行拖拽。
- 拖拽释放后，系统会自动重新计算所有块的索引（Index）。

### 5. 数据类型系统

支持四种数据类型，右键点击 **Value** 区域或边缘即可修改：

- **String** (默认)：普通文本。
- **Number**：数字，会自动处理格式。
- **Boolean**：布尔值 (true/false)。
- **JSON**：嵌套的 JSON 对象或数组。

## 输入与输出

### 输入

- **json_data (Hidden)**: 存储节点状态的 JSON 字符串，通常在界面上隐藏。

### 输出

- **JSON 列表** :
    - 类型：`JSON` (List)
    - 说明：Python 原生列表对象 (`list`)。适合连接到需要列表输入的节点（如批量处理节点）。
